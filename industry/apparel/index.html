02 / 課題

アパレルECが抱えるリピート促進の課題

ISSUE 01

セール頼みの購入サイクルになっている

割引セール時しか購入されないお客様が多く、定価での購入機会が少ない。セール以外の購買動機を作れていない。

ISSUE 02

シーズンをまたいでの継続購入が少ない

春夏コレクションを購入したお客様が、秋冬コレクションでも戻ってくる仕組みがない。年に1〜2回しか接点がないまま終わる。

ISSUE 03

メルマガの開封率低下で連絡が届かない

購入後のフォローメールやメルマガの開封率が下がっており、新作情報がお客様に届きにくくなっています。

ISSUE 04

新規獲得コストが高く、LTVが追いついていない

広告経由の新規獲得コストが上昇する一方、1人あたりのLTVが追いつかず、採算が合わないケースが増えています。

03 / 活用例

アパレルECのキャンペーン企画例

EXAMPLE 01

新シーズン初購入者向けインスタントウィン

対象当シーズン初購入者

期待効果：シーズンをまたいだ継続購入率の向上
EXAMPLE 02

購入金額に応じた特別キャンペーン

対象一定金額以上の購入者
応募同梱タグのQRコードからシリアル入力
賞品豪華ギフト・ブランドノベルティ
参加賞会員プログラム招待・限定クーポン
期待効果：高単価購入の動機づけ・ブランドロイヤルティ強化
EXAMPLE 03

休眠顧客への再購入キャンペーン

対象6ヶ月以上購入のない顧客
応募メールで配信したシリアル入力
賞品大幅割引クーポン・限定商品（抽選）
参加賞復帰感謝クーポン
期待効果：休眠顧客の再活性化・復帰率向上
